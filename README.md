# Homework-00-Student-Grades-Calculator

The program StudentGradesCalculator.java takes the data of 5 students and their grades for 5 subjects. The code takes in all the data and calculates each student's individual average grade from their differnt grades, as well as a class average from all 5 students.

The Student class contains the information for the students that are going to be created including their name and grades. The standard setters and getters, and constructor are included in it, as well as a getAverageGrade() method that is called on in the main class to calculate the average grade of a student once all their grades are inputted.

Once the program is running, you will be asked to enter the name of the first student, and then to enter the grade for each of the 5 different subjects of your choice. This is repeated for the remaining 4 other students, and then all their averages will be delivered back to you in a presentable fashion.
